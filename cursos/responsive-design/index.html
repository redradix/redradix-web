Información básica

Aprende a desarrollar sitios web que se visualicen correctamente en todo tipo de dispositivos. Con una metodología de trabajo ágil, fundamentada en la productividad y la optimización del tiempo.

Duración

20 horas

Nivel

Intermedio

Precio

Normal 525 €

Objetivos

A través de un etiquetado semánticamente estructurado, nuevas reglas de presentación y añadiendo una capa de funcionalidad extra, aprenderás a desarrollar sitios web que se visualicen de manera correcta en todo tipo de dispositivos. Todo esto, con una metodología de trabajo ágil, fundamentada en la productividad y la optimización del tiempo.

Que aprenderemos

  • CSS3

  • SASS

  • Fallbacks
  • Diseño Web Responsive
  • Optimización
  • Mobile First
