Información

React.js, creada por el equipo de Facebook es sin duda alguna la tecnología client-side que más expectación ha generado en los últimos dos años. Su propósito es ayudar a los desarrolladores a crear Single Page Applications con JavaScript que son sencillas de comprender, divertidas de programar y que además ofrecen un gran rendimiento a los usuarios finales.

Junto con React.js, el equipo de Facebook también lanzó en formato código abierto Flux, una arquitectura conceptual para construir SPAs que presenta soluciones originales a los retos habituales del desarrollo front-end.

Objetivos

En este curso aprenderás cómo emplear React.js para construir interfaces de usuario completas para una aplicación Web. También aprenderás sobre Redux, las ideas clave que trae consigo y cómo aplicar esos conceptos al enfocar los requisitos más habituales para cualquier aplicación del mundo real (gestión de datos y estado en el cliente, rutas y comunicación con APIs de servidor).

Requisitos

Este es un curso avanzado dirigido a desarrolladores front-end con experiencia. Para poder cubrir cada tema del curso con la profundidad deseada, esperamos de los alumnos:

  1. Que tengan experiencia trabajando con HTML, CSS y DOM.

  2. Que tengan un nivel medio-alto de JavaScript (funciones, prototipos, clausuras y gestión de “this” en JS, por ejemplo)

  3. Que tengan experiencia con el desarrollo modular utilizando módulos CommonJS (uso de require, exports, etc)

Cualquier experiencia previa con otros frameworks como Backbone.js, Angular o similares ayudará, puesto que te facilitará entender las diferencias y novedades del enfoque ofrecido por React a problemas encontrados habitualmente.
