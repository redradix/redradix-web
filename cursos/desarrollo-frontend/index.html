Información

Aprende los fundamentos que te convertirán en un buen desarrollador front-end. Conocerás cómo utilizar HTML y CSS de una manera ordenada, semántica y escalable.

Duración

20 horas

Nivel

Básico

Precio

525€

Objetivos

El objetivo de este curso es dar las bases para que el alumno se convierta en un desarrollador front-end de calidad. Se verán los fundamentos del lenguaje HTML y su semántica, las reglas de presentación de CSS y una introducción al diseño Responsive. Al finalizar el curso, el alumno entenderá mejor los fundamentos del desarrollo de interfaz y habrá aprendido a utilizar las tecnologías y lenguajes básicos utilizados en su construcción.

Que aprenderemos

  • CSS3
  • JQuery
  • Javascript
  • Diseño Web Responsive
  • HTML5
  • Etiquetas
  • Móvil
  • Semántica

Temario

  1. HTML
    • Definición, reglas y convenciones
    • Sintaxis
    • Categorías de tags
    • Atributos
    • Semántica
    • Modelos de contenido, DOM
    • Herramientas de programación básicas
    • Estandares
  2. CSS
    • Sintaxis y reglas
    • Selectores y unidades de medida
    • Herencia y orden de aplicación
    • Propiedades de texto
    • Propiedades de display y visibilidad
    • Modelo de cajas (box)
    • Tipos y propiedades del layout
    • Backgrounds
    • CSS3 y prefijos de vendedores
    • Decoraciones complejas
    • Transiciones, transformaciones y animaciones
  3. El flujo de trabajo
    • Compatibilidad entre navegadores y mejora progresiva
    • Accesibilidad
    • Testing y debugging
    • Validación de código
  4. Introducción a RWD (Responsive Web Design)
    • Web adaptativa
    • Responsive: media, fuentes y media queries
    • Mejora con JavaScript
